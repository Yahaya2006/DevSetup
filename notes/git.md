# Mes notes sur Git

## Qu'est-ce que Git ?
Un outil qui suit l'historique des modifications d'un projet.

## Commandes essentielles
- `git init` : initialise un dépôt Git
- `git add <fichier>` : prépare un fichier pour le commit
- `git commit -m "message"` : sauvegarde les changements
- `git status` : affiche l'état actuel du dépôt
- `git push` : envoie les commits vers GitHub
