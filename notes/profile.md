# Profil — Page d'accueil

- **Fichier source**: [index.html](index.html)
- **But**: Page de profil / page d'accueil générale présentant l'auteur ou le site.
- **Contenu attendu**: photo de profil, titre, courte biographie, liens vers réseaux sociaux et sections clés du site.
- **Styles**: [styles/style.css](styles/style.css)
- **Images**: dossier `images/` à la racine
- **Prévisualisation**: ouvrir `index.html` dans un navigateur ou lancer un serveur local:

```bash
python -m http.server 8000
# puis ouvrir http://localhost:8000/index.html
```

- **Remarques**: vérifier que les liens relatifs (vers CSS et images) pointent correctement depuis la racine.
